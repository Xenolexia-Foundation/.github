# Xenolexia Foundation

The **Xenolexia Foundation** is a non-profit organization that manages the Xenolexia apps and projects. The Foundation oversees development, ensures consistency across implementations, and maintains the shared vision of making language learning accessible through immersive reading and targeted practice.

---

## Apps & Projects

The following apps and projects are part of the Xenolexia ecosystem (in the parent repository):

| Project | Description |
|--------|-------------|
| [**eBookReader**](https://github.com/Xenolexia-Foundation/eBookReader) | Xenolexia monorepo: Electron desktop app, React Native mobile app, and shared TypeScript core. Learn languages by reading e-books with integrated translation and vocabulary building. |
| [**Grammar-Mini-Trainer**](https://github.com/Xenolexia-Foundation/Grammar-Mini-Trainer) | Grammar mini-trainer for focused practice. |
| [**Listening-Discrimination-Trainer**](https://github.com/Xenolexia-Foundation/Listening-Discrimination-Trainer) | Listening discrimination trainer (e.g. minimal pairs) for desktop and mobile. |
| [**Morphology-Explorer**](https://github.com/Xenolexia-Foundation/Morphology-Explorer) | Local morphology explorer for Arabic and Japanese. Electron and mobile. |
| [**Phrasebook-With-Smart-Search**](https://github.com/Xenolexia-Foundation/Phrasebook-With-Smart-Search) | Offline-first phrasebook with fuzzy search and optional TTS. |
| [**RPG**](https://github.com/Xenolexia-Foundation/RPG) | Language learning RPG built on the Xenolexia shared core (translation, vocabulary, spaced repetition). |
| [**SRS-Flashcard**](https://github.com/Xenolexia-Foundation/SRS-Flashcard) | SRS (spaced repetition) flashcard app for vocabulary review. |
| [**Word-Building-Puzzle**](https://github.com/Xenolexia-Foundation/Word-Building-Puzzle) | Word-building puzzle — Electron (desktop) and React Native (mobile). |
| [**Writing-Coach**](https://github.com/Xenolexia-Foundation/Writing-Coach) | Writing coach for language learners. |
